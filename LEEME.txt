CORRECCIÓN BLACK WEDDING

Este paquete corrige:
- HTML pegado accidentalmente dentro de sections.css
- Iconos SVG demasiado grandes
- Iconos provisionales sustituidos por dresscode.svg, celebration.svg y gifts.svg

Instalación:
1. Haz un commit de respaldo en GitHub Desktop.
2. Copia index.html a la raíz de tu proyecto y reemplázalo.
3. Copia css/sections.css dentro de tu carpeta css y reemplázalo.
4. Confirma que existan:
   assets/icons/dresscode.svg
   assets/icons/celebration.svg
   assets/icons/gifts.svg
5. Abre la invitación y presiona Ctrl + F5.
